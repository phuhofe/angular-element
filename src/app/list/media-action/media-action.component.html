Element

Copy-paste the element code into wherever you want to place it.

<micro-media-list id="media" data-mainColor="{{ mainColor }}" data-secondaryColor="{{ secondaryColor }}"></micro-media-list>

CSS

Copy-paste the stylesheet into your <head> after other stylesheets to load our CSS.

<link rel="stylesheet" href="https://element-demo.web.app/css/styles.css"></link>

<script src="https://element-demo.web.app/js/main.js"></script>
<script src="https://element-demo.web.app/js/4.js"></script>
